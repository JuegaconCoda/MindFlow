☁️ Cuenta en la nube (JSONBin)

Sincroniza tu usuario y mapa entre dispositivos. Gratis en jsonbin.io

Guía rápida JSONBin
  1. Crea cuenta en jsonbin.io → API Keys → copia X-Master-Key
  2. Pégala abajo y pulsa Crear nube (genera el Bin ID)
  3. Pulsa Guardar configuración
  4. En cada dispositivo: misma API Key + Bin ID → inicia sesión con tu usuario

Modo offline (PWA)

En móvil (Android / iPhone):

  1. Publica la carpeta en internet (GitHub Pages, Netlify, o servidor en tu PC).
  2. Abre la URL en Chrome (Android) o Safari (iPhone).
  3. Android: menú ⋮ → Instalar aplicación / Añadir a pantalla de inicio.
  4. iPhone: Compartir ↑ → Añadir a pantalla de inicio.

En PC: usa http://localhost (no abras el .html directo) y pulsa Instalar en la barra de direcciones.
